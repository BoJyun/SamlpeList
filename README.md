# SampleList--樣品排程系統
協助各Admin分配工作與數據整理，並改善工作流程

# 程式架構

# 運作流程
    1.樣品需求者先至網站填寫需求申請並上傳檔案
    2.至In Process網站可以查看排隊流程與是否有申請成功，若完成後可至Complete網站查看
    3.Admin登入後。可以指派工作給旗下的Employer去執行
    4.Admin可以在任何時候查閱各個數據
